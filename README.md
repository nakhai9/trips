# Trips 

Trips là nền tảng xây dựng, quản lý và chia sẻ hành trình du lịch được phát triển bằng **Next.js**, **Spring Boot** và **Material UI**.

## Giới thiệu

Trips là dự án Fullstack giúp người dùng xây dựng các hành trình du lịch, quản lý địa điểm, theo dõi lịch trình và chia sẻ với cộng đồng.

Dự án được xây dựng nhằm thực hành:

* Thiết kế kiến trúc hệ thống Fullstack
* Xây dựng RESTful API
* Xác thực và phân quyền người dùng
* Quản lý dữ liệu với JPA/Hibernate
* Triển khai ứng dụng lên môi trường Production
* Tài liệu hóa API bằng Swagger

---

## Tính năng

### Quản lý hành trình

* Tạo hành trình du lịch
* Chỉnh sửa thông tin hành trình
* Quản lý lịch trình theo từng ngày
* Quản lý địa điểm và hoạt động

### Chia sẻ cộng đồng

* Chia sẻ hành trình với người khác
* Truy cập hành trình thông qua liên kết
* Hỗ trợ lịch trình công khai và riêng tư
* Xem chi tiết lịch trình trên bản đồ

### Trực quan hóa dữ liệu

* Hiển thị lịch trình theo ngày
* Tích hợp bản đồ tương tác
* QR Code chia sẻ hành trình
